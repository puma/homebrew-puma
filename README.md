# Homebrew Puma

Homebrew formula for [https://github.com/puma/puma-dev/releases](https://github.com/puma/puma-dev/releases)

# Installation

## Brewfile
```ruby
# Brewfile
tap 'puma/puma'
brew 'puma/puma/puma-dev'
```

## Manual
```shell
brew install puma/puma/puma-dev
```
